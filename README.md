# 🎨 GesturePaint - Iron Man Edition

GesturePaint is an AI-powered virtual drawing application that allows users to draw on a digital canvas using **hand gestures** and **voice commands**. Inspired by futuristic Iron Man-style interfaces, the project combines Computer Vision, Hand Tracking, and Speech Recognition to create a touchless drawing experience.

---

## 🚀 Features

### ✋ Gesture Controls

* Draw using hand gestures
* Real-time hand tracking
* Smooth freehand painting

### 🎤 Voice Commands

* Change colors using voice
* Switch drawing modes
* Clear canvas
* Undo/Redo actions
* Save drawings

### 🔺 Shape Tools

* Rectangle
* Circle
* Line
* Triangle
* Freehand Drawing

### 🎨 Drawing Options

* Multiple colors
* Adjustable brush size
* Adjustable opacity
* Fill mode
* Eraser tool

### 🎭 Themes

* Dark Theme
* Neon Theme
* Futuristic UI Themes

### ⚡ Additional Features

* Grid Toggle
* Save Artwork
* Undo / Redo Support
* Interactive User Experience

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/Hariom12344/csv.git
cd csv/GesturePaint
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

#### Windows

```bash
.venv\Scripts\activate
```

#### Linux / Mac

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Application

```bash
python main.py
```
